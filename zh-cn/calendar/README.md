# 华炎云日历
#### 什么是华炎云日历？
华炎云日历是由上海华炎软件公司开发的一款云日历。

它不但涵盖了传统日历的众多功能，如时间查阅、时区设置及事件记录等，而且还推出了自己的**特色功能，如日历、事件的共享与同步功能**。有了华炎云日历，
- 您可以将自己的日程安排分享好友或同事，减少了繁冗的口头传达，极大地缩减了沟通成本。
- 此外，您还可以将日程安排同步到您的移动设备，随时随地进行查阅。

接下来就为您介绍华炎云日历的使用方法。
